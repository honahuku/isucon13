# isucon13
## ビルド
```bash
cd webapp/go/
go build -o isupipe && sudo systemctl restart isupipe-go.service
```
## ログ
```bash
# リアルタイム表示
sudo journalctl -f -u isupipe-go.service

# 末尾から30行表示
sudo journalctl -u isupipe-go.service -n 30
```
